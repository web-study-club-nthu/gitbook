#Typesetting
* Division: `<div>`
* Inline element: `<span>`
* HTML Frame: `<iframe>`
    * Source attribute: `src`
    * Size attributes: `height`, `width`
* Difference:
    * `<div>` is more like a **block**
    * `<span>` is in the line
    * `<iframe>` is a separate HTML file
```
<iframe src="http://heyou.tw/" height="500px" width="1000px">
```
